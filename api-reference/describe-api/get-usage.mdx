Check remaining AI credits without generating a logo.

## Request

<ParamField header="Authorization" type="string" required>
  `Bearer <api_key>` - Studio API key (`ll_live_…`).
</ParamField>

<RequestExample dropdown>

```typescript TypeScript
interface UsageResponse {
  monthlyUsed: number;
  monthlyLimit: number;
  monthlyRemaining: number;
  bonusCredits: number;
  remaining: number;
  limit: number;
  resetsAt: string;
}

const response = await fetch("https://logolattice.com/api/v1/logos/usage", {
  headers: {
    Authorization: `Bearer ${process.env.LOGO_LATTICE_API_KEY!}`,
  },
});

const data = (await response.json()) as UsageResponse;
console.log(data.remaining, data.resetsAt);
```


```python Python
import os
import requests

response = requests.get(
    "https://logolattice.com/api/v1/logos/usage",
    headers={
        "Authorization": f"Bearer {os.environ['LOGO_LATTICE_API_KEY']}",
    },
    timeout=30,
)

data = response.json()
print(data["remaining"], data["resetsAt"])
```

```php PHP
<?php

$apiKey = getenv('LOGO_LATTICE_API_KEY');

$ch = curl_init('https://logolattice.com/api/v1/logos/usage');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
    ],
]);

$raw = curl_exec($ch);
curl_close($ch);

$data = json_decode($raw, true);
echo $data['remaining'] . PHP_EOL;
echo $data['resetsAt'] . PHP_EOL;
```

## Response

<ResponseField name="monthlyUsed" type="integer">
  Credits used this calendar month (UTC).
</ResponseField>

<ResponseField name="monthlyLimit" type="integer">
  Monthly allowance. Studio includes **10** generations per calendar month (UTC), shared with in-editor Describe and AI stylise.
</ResponseField>

<ResponseField name="monthlyRemaining" type="integer">
  Credits left in the monthly pool before bonus credits.
</ResponseField>

<ResponseField name="bonusCredits" type="integer">
  Extra credits from [$5.99 top-ups](https://logolattice.com) (30 credits per purchase). Added to `remaining`.
</ResponseField>

<ResponseField name="remaining" type="integer">
  Total credits available for generation.
</ResponseField>

<ResponseField name="limit" type="integer">
  Deprecated - prefer `monthlyLimit` for display.
</ResponseField>

<ResponseField name="resetsAt" type="string">
  ISO 8601 timestamp when the monthly pool resets (UTC month boundary).
</ResponseField>

## HTTP status codes

| Status | When                                      |
| ------ | ----------------------------------------- |
| `200`  | Success                                   |
| `401`  | Missing, invalid, or revoked Bearer token |
| `403`  | Account is not Studio                     |

Auth errors use the same shape as [Generate logo](/api-reference/describe-api/generate-logo). See [Errors](/api-reference/describe-api/errors).
