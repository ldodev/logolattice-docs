title: Errors
description: HTTP status codes and error codes for the Logo Lattice API

---

Failed requests return JSON:

```json
{
  "error": "Human-readable message",
  "code": "error_code",
  "usage": { },
  "details": { }
}
```

- `code` - machine-readable when present
- `usage` - included when relevant (for example quota errors), same fields as a successful generation
- `details` - Zod validation (for example `fieldErrors`)

## HTTP status codes

| Status | When |
| ------ | ---- |
| `200` | Success |
| `400` | Invalid JSON; Zod validation (`details` with `fieldErrors`); `invalid_input`; `parse_error`; `project_limit_exceeded` (Describe only) |
| `401` | `invalid_key` or `revoked` |
| `403` | `not_premium` - not on Studio |
| `429` | `limit_exceeded` - no credits left (`usage` included) |
| `502` | `upstream` - provider failure |
| `503` | `not_configured` - AI/SVG provider not configured on server |

## Error codes

| Code | HTTP | Meaning |
| ---- | ---- | ------- |
| `invalid_key` | `401` | Bad or missing API key |
| `revoked` | `401` | Key was revoked |
| `not_premium` | `403` | Studio subscription required |
| `invalid_input` | `400` | Bad reference image, prompt rules, etc. |
| `parse_error` | `400` | Model returned empty or unusable logo (Describe) |
| `project_limit_exceeded` | `400` | Cloud project cap hit (Describe only) |
| `limit_exceeded` | `429` | No AI generations left |
| `upstream` | `502` | Provider failure |
| `not_configured` | `503` | AI/SVG provider not configured |

## Example: validation

```json
{
  "error": "Invalid request.",
  "details": {
    "fieldErrors": {
      "prompt": ["String must contain at least 10 character(s)"]
    }
  }
}
```

Other **400** cases include invalid hex in `brandColors`/`referenceColors` and `brandName` longer than 64 characters after trim.

## Example: quota exceeded

```json
{
  "error": "You have no AI generations left. Top up for 30 more, or wait until your free quota resets.",
  "code": "limit_exceeded",
  "usage": {
    "monthlyUsed": 10,
    "monthlyLimit": 10,
    "monthlyRemaining": 0,
    "bonusCredits": 0,
    "remaining": 0,
    "limit": 10,
    "resetsAt": "2026-07-01T00:00:00.000Z"
  }
}
```

<Tip>
  Poll `GET /api/v1/logos/usage` before batch jobs to avoid `429` responses mid-run.
</Tip>
