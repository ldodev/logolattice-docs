## v1.0.0

<Note>Initial public release of the Studio Logo API under `/api/v1`.</Note>

- `POST /api/v1/logos` - generate SVG and cloud project from a text prompt
- `GET /api/v1/logos/usage` - read AI credit balance without generating
- Bearer authentication with `ll_live_` API keys
- Optional reference images (PNG, JPEG, WebP; max 4 MB decoded)

Future breaking changes will ship under a new version prefix (for example `/api/v2`). This page will list deprecations and migration notes as they are announced.
